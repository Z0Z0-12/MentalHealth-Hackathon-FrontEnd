# Manasly 🌿

A mental health and student wellbeing platform built for university students. Manasly brings together housing, careers, community events, forums, and peer chat — all in one place.

---

## Features

- **Housing** — Browse apartment listings near campus with photos, rent, beds/baths, and contact info
- **Careers** — Explore internships and job opportunities tailored for students
- **Events** — Discover and RSVP to upcoming university and community events
- **Forum** — Ask questions, share experiences, and connect with peers
- **Chat** — Send direct messages to other students with real-time WebSocket support
- **Anonymous mode** — Post and chat anonymously for sensitive topics

---

## Tech Stack

**Frontend**
- React + Vite
- Tailwind CSS
- Lucide React
- React Router

**Backend**
- FastAPI (Python)
- PostgreSQL
- Keycloak (JWT authentication)
- Hosted on Railway
